# SCI Enhanced Resources Process

## 1.0.0 (2024-10-29)
- 初期リリース

## 1.1.1 (2025-09-25)
- 案件情報のグループ化、表示順設定を可能にしました。
- 案件情報マスタメンテナンスを実装しました。
- アップロード時にグループID・表示順は初期値を設定します。

## 1.2.0 (2025-10-29)
- カンパニーカラーに合わせて、白黒基調に変更しました。
- Microsoft Azure認証機能を追加しました。
- ログイン後にメニュー画面を表示するようフローを変更しました。
- ログイン機能の追加により、各テーブルに更新者を保存するよう変更しました。
- メニュー画面にリリースノートを表示するよう機能を追加しました。
- その他、軽微な修正対応を実施しました。
